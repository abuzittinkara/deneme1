UI Bileşenleri Demo

Bu sayfa, kullanıcı arayüzü bileşenlerini göstermek için bir demo sayfasıdır.

Toast Bildirimleri

Kullanıcıya bilgi vermek için toast bildirimleri kullanabilirsiniz.

UI.showToast('Mesaj', 'success|error|warning|info', 3000);

Modal Dialog

Kullanıcıdan bilgi almak veya bilgi göstermek için modal dialog kullanabilirsiniz.

UI.showModal({
  title: 'Modal Başlığı',
  content: 'Modal içeriği',
  buttons: [
    { text: 'İptal', value: 'cancel', type: 'secondary' },
    { text: 'Tamam', value: 'ok', type: 'primary' }
  ],
  closeOnBackdrop: true,
  size: 'medium'
}).then(result => {
  console.log('Modal sonucu:', result);
});

Onay Dialog'u

Kullanıcıdan onay almak için onay dialog'u kullanabilirsiniz.

UI.showConfirm('Bu işlemi gerçekten yapmak istiyor musunuz?', 'Onay')
  .then(result => {
    if (result) {
      UI.showToast('İşlem onaylandı!', 'success');
    } else {
      UI.showToast('İşlem iptal edildi.', 'info');
    }
  });

Uyarı Dialog'u

Kullanıcıya uyarı göstermek için uyarı dialog'u kullanabilirsiniz.

UI.showAlert('Bu bir uyarı mesajıdır!', 'Uyarı')
  .then(() => {
    UI.showToast('Uyarı kapatıldı.', 'info');
  });

Giriş Dialog'u

Kullanıcıdan metin girmesini istemek için giriş dialog'u kullanabilirsiniz.

UI.showPrompt('Lütfen adınızı girin:', '', 'Giriş')
  .then(result => {
    if (result) {
      UI.showToast(`Merhaba, ${result}!`, 'success');
    } else {
      UI.showToast('Giriş iptal edildi.', 'info');
    }
  });

Yükleniyor Göstergesi

Uzun süren işlemler için yükleniyor göstergesi kullanabilirsiniz.

const loading = UI.showLoading('Yükleniyor...');

// 1 saniye sonra mesajı güncelle
setTimeout(() => {
  loading.updateMessage('Veriler alınıyor...');
}, 1000);

// 2 saniye sonra mesajı güncelle
setTimeout(() => {
  loading.updateMessage('İşlem tamamlanıyor...');
}, 2000);

// 3 saniye sonra yükleniyor göstergesini kapat
setTimeout(() => {
  loading.hide();
  UI.showToast('İşlem tamamlandı!', 'success');
}, 3000);
